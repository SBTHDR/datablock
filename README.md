# DataBlock


## About Repository

DataBlock - A Basic Data Science Job Listing System built with Laravel 8 where user can post a job listing by paying through Stripe. 


## Tech 

- Laravel 8
- Tailwind CSS
- Stripe Elements


## Features

- User Listing CRUD
- Login, Register using Breeze auth
- Search listing
- Sort by Tag
- User Agent/IP Store


## Installation

- `git clone https://github.com/SBTHDR/datablock.git`
- `cd datablock/`
- `composer install`
- `cp .env.example .env`
- Update `.env` and set your database credentials
- `php artisan migrate`
- `php artisan db:seed`
- `npm install`
- `npm run dev`
- `php artisan serve`
